# Prompt2Repo 入门考核 (Mini Challenge)

## 🎯 考核目标
利用 AI 工具快速构建一个**包含前后端**的 Web 应用，并实现 Docker 一键启动。

## 🛠 题目要求
1. **功能**：实现一个简单的 [你的题目，如：图书管理系统]。
2. **技术栈**：React/Vue + Python/Node.js + SQLite。
3. **红线标准**：
   - 必须包含 `docker-compose.yml`。
   - 界面需具备基本的 UI 审美（推荐使用组件库）。

## 🚀 提交方式
1. 点击本仓库右上角的 **"Use this template"** 将题目复制到你的账号下。
2. 完成代码编写，并 Push 到你的仓库。
3. **自测**：确保你仓库的 **Actions** 页面显示为 ✅ (Green)。
4. 将你的 **仓库链接** + **运行录屏** 提交给招聘方。
